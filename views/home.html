Welcome


This site was built as a learning project to learn web development. I had a need to test and use some Microsoft Azure B2C APIs. Once I figured out how the API worked, I started thiking of ways the API could be built into a web app, and so my adventure began.


Through the process of building this site I have learned how to use and implement the techologies below, which have all been used in conjection to make this site function.


Node JS

As an asynchronous event-driven JavaScript runtime, Node.js is designed to build scalable network applications. Many connections can be handled concurrently. Upon each connection, the callback is fired, but if there is no work to be done, Node.js will sleep.

Pug

Pug is a high-performance template engine heavily influenced by Haml and implemented with JavaScript for Node.js and browsers.

Bulma

Bulma is a free, open source CSS framework based on Flexbox and used by more than 200,000 developers.
